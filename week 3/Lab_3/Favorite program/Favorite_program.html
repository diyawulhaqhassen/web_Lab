Garden Invoice

Garden invoice program generate invoice.
The invoices have the following information:

  • The customer name
  • Address
  • Date of service
  • The size of the garden
  • The mowing charge
  • The leaf raking charge
  • The total for all services
  • Contact information for the gardener
The following are sample code 

    public static boolean writeToFile(String filename, String text) {

        try (BufferedWriter writer = new BufferedWriter(new FileWriter(new File(INVOICE_DIRECTORY, filename)))) {
            writer.write(text);
            writer.close();
            return true;

        } catch (IOException e) {
            System.out.println("Unable to write to file " + filename + ". Error message:\n" + e.getMessage());
            return false;
        }
    }
